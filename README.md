# Employee Management System

Application for management employees write in Django


## Getting Started

1. Create a virtualenv
2. Install Django 3.0
3. Install requirements.txt - pip install -r requirements.txt
4. Run the server

## Built With

* Django 
* JavaScript
* HTML
* CSS
